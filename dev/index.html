Available Commands

Build only the core library

npm run build:cli

Build only the CLI tools

Framework Structure

  • Core: Main presentation engine
  • CLI: Command-line interface tools
  • Templates: Ready-to-use presentation templates
  • Themes: Built-in color themes
  • Plugins: Extensible plugin system

Quick Start

To create a new presentation:

node dist/cli/index.js example --template basic

Then run:

node dist/cli/index.js dev

Or for specific directory:

node dist/cli/index.js dev -d /path/to/project
